====================
Embed YouTube videos
====================

.. rst-class:: bignums

1. Get the YouTube id.

   Example: The URL is `https://www.youtube.com/watch?v=rWkVS9qxBmg`.

   The id is the text after `v=` and up to the next `&` or the end of the
   URL.

   Here: `rWkVS9qxBmg`.

2. Embed the Video using the id


   .. code-block:: rest
